# Multi-Router Administrative Distance Project (Cisco Packet Tracer)

A live network simulation project implemented in Cisco Packet Tracer, demonstrating static routing, subnetting, 
and using Administrative Distance to Send a Packet to Clock Wise Direction form Sending Packet to Diff Router and Reciver from Diff Router.
